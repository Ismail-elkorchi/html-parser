# Support

## Security issues
Report security vulnerabilities using the process in `SECURITY.md`.

## Usage questions and bug reports
Open a GitHub issue with:
- runtime and version details
- minimal reproduction input
- expected vs actual behavior
- relevant command output (`npm run eval:ci` or `npm run eval:release` when applicable)

## Scope reminders
This project is an HTML parser and serializer for deterministic automation workflows. It is not a DOM runtime or sanitizer.
